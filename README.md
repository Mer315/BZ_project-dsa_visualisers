# Simple DSA Visualisers
 ### LRU Cache visualiser
    - takes user input
    - represents visualisation using nodes
    - displays description for each step being performed
 ### Sorting Algorithms
    visualises the following alogrithms
    - Selection sort
    - Insertion sort
    - Bubble Sort
 ### AVL Tree
    - visualises the construction of an avl tree and does the following operations 
        - insert
        - delete
        - search
    - also provides description of every step performed with proper depiction of the tree formed
